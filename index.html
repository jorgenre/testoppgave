Twitter archive

The task is to search Twitter with JavaScript. Normally we make a call to a microservice, that returns json. But in this case we get the results from static json https://www.retriever-info.com/oppgave/twitter.json.

The post you get from twitter.json should be listed in <div id="result"> with the field "story" and "author.displayName". It's up to you how to use the other fields.

It should be possible to filter resultset by write author name in the input field.

You choose the usages of framework. Style the list as you want, it should be easy to get a quick overview. You can do the task in your own file, if you do not want to reuse this form.
